# This file lets you extend Horde's menu with your own links and graphics
#
# The format of this file is as follows:
#
# URL<Linked Text>Image File
#
# Where:
#
#  o URL         = the URL to link to
#  o Linked Text = the text to display
#  o Image File  = The graphic to display (found in $default->graphics_url/<Image File>)
#
# An example of this follows (its commented out)
#
http://www.debian.org/<Debian>search.gif
